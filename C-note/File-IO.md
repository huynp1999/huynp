Các hàm xử lý file trong thư viện C:

| Hàm | Mô tả |
| ------------- |:-------------:|
| fopen()      | right foo     |
| fprintf()      | right bar     |
| fscanf()      | right baz     |
|  fputc()     |  ghi một ký tự vào file   |
|  fgetc()     |  đọc một ký tự từ file   |
|    fclose()   |    đóng file |
|   fseek()    |  chỉ định vị trí con trỏ tập tin   |
|   rewind()    |  đặt con trỏ tập tin vào đầu tập tin   |
